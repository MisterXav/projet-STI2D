Mini-projet de Terminale STI2D - SIN

  Ce site est le résultat d'un mini-projet réalisé par des élèves de Terminale STI2D - SIN.

Le but de ce projet est de permettre aux élèves de mobiliser leurs connaissances sur la gestion d'un site web dynamique. Pour cela, il leur a été demandé de concevoir un site permettant d'effectuer différents calculs :

  • Périmètre et aire d'un rectangle en fonction de ses longueur et largeur
  • Résolution d'une équation 1er degré : ax + b = 0
  • Résolution d'une équation 2nd degré : ax2 + bx + c = 0
  • Convertir un nombre dans les bases 10 (décimal), 2 (binaire) et 16 (hexadécimal)

Choisissez votre calcul à l'aide du menu de gauche.
